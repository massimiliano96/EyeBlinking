# Requirements
- Opencv (version > 4.x)
    ```
    sudo apt install libopencv-dev python3-opencv
    ```
- Cmake (version > 3.22.x)
    ```
    sudo apt install cmake
    ```
- gTest (version > 1.11.x)
    ```
    sudo apt-get install libgtest-dev
    cd /usr/src/gtest
    sudo make
    sudo cp lib/*.a /usr/lib
    ```
